
This project uses Python and data analytics to predict student performance based on exam scores and attendance data. The goal is to help educators identify at‑risk learners early and provide targeted support.

## Features
- Data cleaning and preprocessing with **Pandas**
- Exploratory Data Analysis (EDA) with **Matplotlib/Seaborn**
- Predictive modeling using **scikit‑learn**
  


## Dataset
The dataset includes:
- Student IDs
- Subject scores (Math, Science, English)
- Final grade (target variable)
- Attendance rates
